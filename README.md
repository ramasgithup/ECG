# ECG
# Integrating Explainable and Green AI Models for ECG Image Diagnosis 

This project develops an **interpretable** 🧠 and **energy-efficient** ⚡ AI model for automated ECG image diagnosis. The model is designed to be **fast, accurate, and explainable**, supporting decision-making in various settings:

- **🏥 Hospitals:** Supports fast and interpretable ECG analysis.  
- **🌐 Telemedicine:** Enables ECG assessment in remote areas.  
- **📚 Education & Research:** Provides transparent AI-based explanations for learning and research purposes.
